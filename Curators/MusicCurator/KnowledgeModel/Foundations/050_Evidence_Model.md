# 050 – Evidence Model

Status: Draft  
Version: 0.1

## Definition

Evidence describes the basis of an assertion.

## Examples

- MusicBrainz record
- Discogs record
- RISM entry
- Grove article
- Autograph
- Catalogue
- User observation
- Imported metadata
- AI inference

## Rule

Evidence supports an assertion.  
It does not itself become truth.
