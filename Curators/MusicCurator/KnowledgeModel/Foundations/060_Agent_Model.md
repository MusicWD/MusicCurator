# 060 – Agent Model

Status: Draft  
Version: 0.1

## Definition

An Agent is an actor that makes, records, imports, confirms, rejects, or modifies assertions.

## Examples

- Wolfgang
- MusicBrainz
- Discogs
- RISM
- Grove
- AI assistant
- Import script
- Application user

## Rule

Assertions should remain traceable to their agent wherever possible.
