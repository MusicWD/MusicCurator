# 020 – Entity Model

Status: Draft  
Version: 0.1

## Definition

An Entity is any real, abstract, conceptual, cultural, temporal, or material object that can be the subject of assertions or relations.

## Initial Entity Categories

- Person
- Work
- Expression
- Manifestation
- Item
- Place
- Event
- Organisation
- Collection
- Concept
- Object
- TimeSpan

## Rule

Entity categories are not arbitrary tags.  
They must be introduced only when they change modelling behaviour, validation, or interpretation.
