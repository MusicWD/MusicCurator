# Relation Vocabulary

Status: Draft  
Version: 0.1

## Rule

Relation qualifiers are finite, controlled, and extendable only by documented decision.

Each qualifier requires:
- Identifier
- Label
- RelationClass
- Definition
- Direction
- Allowed source types
- Allowed target types
- Examples
- Counterexamples

## Identity

- same_as
- alias_of
- spelling_variant_of
- translated_title_of
- former_name_of

## Structural

- part_of
- contains
- movement_of
- chapter_of
- version_of
- belongs_to_group

## Semantic
