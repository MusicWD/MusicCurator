# 030 – Relation Model

Status: Draft  
Version: 0.1

## Definition

A Relation connects two entities.

Relations are first-class model objects.

## Relation Fields

- sourceEntity
- targetEntity
- relationClass
- qualifier
- direction
- scope
- evidence
- confidence
- status

## Relation Classes

- Identity
- Structural
- Semantic

## Rule

A relation connects entities.  
It does not automatically transfer assertions between them.
