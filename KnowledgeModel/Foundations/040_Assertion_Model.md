# 040 – Assertion Model

Status: Draft  
Version: 0.1

## Definition

An Assertion is a statement made about a subject.

## Structure

- subject
- predicate
- object
- agent
- evidence
- confidence
- scope
- validity
- status

## Consequence

Traditional attributes are treated as assertions.

Examples:
- Title is an assertion.
- Genre is an assertion.
- Composer is an assertion.
- Language is an assertion.
- Date is an assertion.
